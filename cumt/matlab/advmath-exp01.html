第四题

已知制作一个背包的成本为 40 元. 如果每一个背包的售出价为 xxx 元，售出的背包数为

n=m+1x−40+(m+1)(80−x)n = \frac{m+1}{x-40} + (m+1)(80-x)n=x−40m+1​+(m+1)(80−x)

问什么售出价格能带来最大利益？（40≤x≤8040 \leq x \leq 8040≤x≤80）

syms x
n(x) = (m + 1)/(x - 40) + (m + 1)*(80 - x);
profit(x) = n(x) * (x - 40);
[x_opt, neg_max_profit] = fminbnd(matlabFunction(-profit(x)), 40, 80);
max_profit = -neg_max_profit;

x_opt
max_profit
